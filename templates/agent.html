Comment ça marche

L'agent teste différentes combinaisons d'hyperparamètres :

  • Grid Search : Teste toutes les combinaisons
  • Random Search : Teste des combinaisons aléatoires

Chaque combinaison est entraînée sur un sous-ensemble des données pour être plus rapide.

L'optimisation peut prendre plusieurs minutes selon le nombre d'essais.
